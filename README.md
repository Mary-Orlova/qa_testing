**GET /pet/{id} возвращает 200 OK с некорректными данными (пустое обязательное поле photoUrls)**

Шаги воспроизведения:
 - Отправить GET запрос на /pet/{petId} (id=1, 2, 3, 6, 11);
 - Проверить тело ответа, в частности, поле photoUrls.

Ожидаемый результат: 200 OK. В теле ответа поле photoUrls должно содержать непустое значение

Фактический результат: 200 OK
В теле ответа поле photoUrls пустое ([]), хотя в схеме Swagger оно помечено как обязательное (*).

Серьезность: Medium

Приоритет: High
